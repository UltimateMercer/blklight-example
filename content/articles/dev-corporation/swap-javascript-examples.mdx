Há três maneiras comuns de fazer o swap de duas variáveis em JavaScript:

<Tabs defaultValue="1" className="w-full">
  <TabsList>
    <TabsTrigger value="1">1</TabsTrigger>
    <TabsTrigger value="2">2</TabsTrigger>
    <TabsTrigger value="3">3</TabsTrigger>
  </TabsList>
  <TabsContent value="1">

    Usando uma variável temporária:

    ```js
      let x = 10;
      let y = 20;

      let temp = x;
      x = y;
      y = temp;
    ```

  </TabsContent>
  <TabsContent value="2">
    
    Usando a aritmética de desestruturação:

    ```js
      let x = 10;
      let y = 20;

      [x, y] = [y, x];
    ```

  </TabsContent>
  <TabsContent value="3">

    Usando operações bit-a-bit:

    ```js
      let x = 10;
      let y = 20;

      x = x ^ y;
      y = x ^ y;
      x = x ^ y;
    ```

  </TabsContent>
</Tabs>
