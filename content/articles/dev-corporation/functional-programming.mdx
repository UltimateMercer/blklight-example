A programação funcional é um paradigma de programação que enfatiza a composição de funções matemáticas para resolver problemas. Em vez de modificar estados ou mudar o estado do sistema, as funções puras em programação funcional são idempotentes e retornam sempre o mesmo resultado dado os mesmos argumentos.

Aqui estão alguns exemplos de linguagens de programação que suportam programação funcional:

- Haskell: Uma linguagem de programação funcional pura, desenvolvida para fins acadêmicos. É usada para fins de pesquisa e ensino, bem como para desenvolvimento de aplicativos.

- Lisp: Uma linguagem de programação funcional antiga, com um longo histórico de uso em inteligência artificial e pesquisa.

- Scheme: Uma variante de Lisp que é amplamente utilizada em cursos de programação funcional.

- Erlang: Uma linguagem de programação funcional criada para a construção de sistemas distribuídos confiáveis.

- F#: Uma linguagem de programação funcional da Microsoft, que combina características de programação funcional e orientada a objetos.

- Scala: Uma linguagem de programação multi-paradigma, que combina programação funcional e orientada a objetos.

- JavaScript: Embora JavaScript não seja uma linguagem de programação funcional pura, ele suporta características funcionais, como funções anônimas e encadeamento de funções.

Estes são apenas alguns exemplos de linguagens de programação funcional. Cada uma tem suas próprias características únicas e é importante considerar suas necessidades específicas ao escolher a linguagem apropriada para o seu projeto.
