Uma closure em JavaScript é uma função que se lembra de seu escopo léxico, ou seja, onde foi definida, mesmo quando é invocada fora desse escopo. Em outras palavras, uma closure é uma função que mantém uma referência ao seu contexto de criação, incluindo as variáveis e funções que estavam disponíveis no escopo léxico na época da sua criação.

Aqui está um exemplo simples de closure:

```js
function outerFunction(x) {
  return function innerFunction(y) {
    return x + y;
  };
}

const add5 = outerFunction(5);
console.log(add5(3)); // Output: 8
```

Neste exemplo, a função outerFunction retorna a função innerFunction, que tem acesso ao argumento x passado para outerFunction, mesmo quando é invocada fora de seu escopo. Quando a variável add5 é definida como o resultado da invocação de outerFunction(5), ela armazena uma referência a innerFunction com o argumento x igual a 5. Portanto, quando add5 é invocada com o argumento 3, o valor retornado é 8.
